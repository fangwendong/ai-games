# 种花游戏

一个浏览器可玩的空间逻辑谜题原型。方向参考《Meowdoku!》这类产品公式：成熟网格推理、轻主题包装、短局体验、低门槛交互。

## 运行

进入本目录后执行：

```bash
python3 -m http.server 4174
```

然后打开 `http://127.0.0.1:4174/`。

这是单文件静态网页，直接打开 `index.html` 通常也能运行。

## 操作

- 默认是 `种花` 模式，点击格子会种花或移走花
- 切到 `排除` 模式，再点击格子：标记这里不可能有花
- 双击格子：桌面端快捷种花
- `玩法`：打开规则说明弹窗
- `H`：显示一个提示
- `B`：自动种下一朵正确的花，整轮最多 3 次
- `R`：重开当前关
- `N`：当前关完成后进入下一关；连续通过第 10 关后重新开一轮
- `1`：切到排除模式
- `2`：切到种花模式

## 当前状态

第一版可玩原型：

- 100 个确定性生成关卡作为题库
- 当前规则是 30 连闯：1-10、10-20、20-30 三段，全部连续通过才算通关
- 1-10 段给 3 条命，10-20 段给 6 条命，20-30 段给 9 条命
- 失败只会回到当前段起点，不会回到第 1 关
- 整轮累计 3 次开花机会，过单关不恢复
- 单关通过、段落通过、30 关通关、闯关失败都会先弹出提示，不会直接跳走
- 单关通过显示高兴表情，闯关失败显示沮丧表情
- 行、列、花圃、不相邻规则校验
- 显式点击模式，降低新手理解成本
- `种花 / 排除` 放在主界面最上方，默认种花
- 玩法说明收进弹窗，不占用主棋盘空间
- 提示和自动种花道具
- 10 关连续闯关进度条
- 关卡进度和轻量状态面板
- 适配桌面和移动端布局
