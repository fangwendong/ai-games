# 密室逃脱

一个浏览器可玩的分关密室逃脱原型。每一关都是一个独立房间，玩家需要从现场找到线索，推导密码或开锁方式，打开门后进入下一关。

## 运行

进入本目录后执行：

```bash
python3 -m http.server 4175
```

然后打开 `http://127.0.0.1:4175/`。

这是单文件静态网页，直接打开 `index.html` 通常也能运行。

## 操作

- 点击房间里的物件，查看线索
- 点击门锁输入密码
- 部分关卡需要找到开锁方式而不是纯数字密码
- `提示`：显示当前关的提示
- `重置本关`：清空当前关状态
- `下一关`：通关后进入下一关

## 当前状态

第一版可玩原型：

- 8 个关卡
- 每关一个独立房间主题
- 物件点击探索
- 密码锁、开关锁、顺序锁、字母锁
- 每关都有明确的过门条件
- 通关和失败反馈
- 适配桌面和移动端
